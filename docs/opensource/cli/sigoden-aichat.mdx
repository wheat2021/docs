## 概述

AIChat 是一款功能强大的全能命令行界面（CLI）工具，旨在改变您与大语言模型（LLMs）的交互方式。该工具采用 Rust 构建，兼顾性能与可靠性；通过统一接口无缝集成 20 多家 LLM 提供商，成为终端中 AI 驱动工作流的多功能助手。

AIChat 将终端转变为智能工作空间：您可以通过多种交互模式与 AI 对话、自动化任务并提升效率。无论是开发者、系统管理员还是高级用户，AIChat 都能凭借灵活架构与完善功能适配您的工作流。

## 交互模式

AIChat 提供三种针对不同用例的交互方式：

<Columns>
  <Card title="CMD 模式">
    直接在命令行完成单次 AI 交互，适合脚本集成与快速问答，无需进入会话即可得到结果。
  </Card>
  <Card title="REPL 模式">
    交互式 Chat-REPL（读取-求值-打印循环），支持补全、多行输入、历史搜索、可配置按键绑定与自定义提示词，持续保留上下文。
  </Card>
  <Card title="Shell 助手">
    使用自然语言描述意图，自动生成精确的 shell 命令，适配您的系统与 shell 环境，兼顾入门与高效。
  </Card>
</Columns>

## 核心架构

AIChat 基于模块化架构构建，在保持组件紧密集成的同时实现关注点分离：

该架构围绕几个关键模块协同工作：

- **客户端模块** (`src/client/`)：统一接口对接多家 LLM 提供商，支持 OpenAI、Claude、Gemini 等，包含原生与 OpenAI 兼容端点。
- **配置系统** (`src/config/`)：管理模型选择、角色定义、会话管理与用户偏好。
- **REPL 引擎** (`src/repl/`)：提供交互式 shell 体验，具备命令补全、语法高亮与会话管理。
- **RAG 实现** (`src/rag/`)：集成外部文档实现检索增强生成，提升准确性与上下文相关性。
- **服务器组件** (`src/serve/`)：提供 HTTP 服务，可作为带 API 与 Web 界面的服务运行。

## 多提供商支持

AIChat 对 LLM 提供商具备广泛支持，集成包括：

- **主要提供商**：OpenAI、Claude (Anthropic)、Gemini (Google AI Studio)
- **自托管**：Ollama、本地 OpenAI 兼容服务器
- **云平台**：Azure OpenAI、AWS Bedrock、Google Vertex AI
- **专业服务**：Groq、Perplexity、Cohere、Mistral、DeepSeek
- **开发者平台**：GitHub Models、AI21、XAI Grok

多提供商支持通过客户端注册系统实现，便于扩展新提供商并为上层提供一致 API。

## 高级功能

<Tabs>
  <Tab title="角色系统">
    通过将提示词与模型配置组合，定义面向特定任务的角色。例如设定专精代码质量分析的“代码审查员”，或擅长文档编写的“技术写作”角色。
  </Tab>
  <Tab title="会话管理">
    维护上下文感知的多轮对话，保留历史以便引用过往内容，确保交互连续性与一致性。
  </Tab>
  <Tab title="RAG">
    集成外部文档到 LLM 对话中，基于嵌入进行语义检索，并可提供引用来源以增强可验证性。
  </Tab>
  <Tab title="函数调用与工具">
    将 LLM 连接到外部工具与数据源（如网页搜索、文件操作、API 调用），把能力从文本生成扩展到实际任务自动化。
  </Tab>
  <Tab title="AI 代理">
    将指令、工具与文档组合为可处理复杂工作流的代理，类似于在命令行中构建具备特定能力与知识库的助手。
  </Tab>
  <Tab title="Web 服务器功能">

| 能力 | 说明 |
|---|---|
| 聊天补全 API（OpenAI 格式） | 以 OpenAI 兼容接口提供对话能力 |
| 嵌入 API | 提供向量化嵌入以支持语义操作 |
| 重排序 API | 用于文档候选的相关性重排序 |
| 浏览器交互 Playground | Web 界面的交互式 LLM 游乐场 |
| 模型对比 Arena | 并排比较不同模型的输出 |

  </Tab>
</Tabs>

## 入门指南

<Steps>
  <Step title="安装">

```bash
cargo install aichat

# 或使用包管理器
brew install aichat          # macOS
pacman -S aichat             # Arch Linux
scoop install aichat         # Windows
```

  </Step>
  <Step title="首次使用">

```bash
# CMD 模式快速提问
aichat explain quantum computing

# 启动交互式 REPL 模式
aichat

# 使用 shell 助手
aichat -e "find all large files in current directory"
```

  </Step>
</Steps>

## 配置与自定义

AIChat 可通过 YAML 配置文件、环境变量与命令行选项进行高度配置；首次运行会生成配置文件，可自定义：

- 默认模型与提供商设置
- API 密钥与身份验证
- 角色定义与提示词
- RAG 配置与文档路径
- 函数调用与工具集成
- UI 主题与显示选项

## 为什么选择 AIChat？

AIChat 的优势包括：

1. **统一多种接口**：在一个工具中整合 CMD、REPL 与 shell 助手模式
2. **广泛提供商支持**：通过统一接口访问主要 LLM 提供商
3. **高级功能齐全**：RAG、函数调用、代理与 Web 服务能力
4. **高性能实现**：Rust 构建，性能与稳定性兼顾
5. **灵活可扩展**：高度可配置、便于扩展，适配多类用例
