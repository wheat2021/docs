## Zeabur 是什么？

Zeabur 是一个新一代的云端部署平台，主打“一键部署、自动扩容、多语言支持”，主要面向开发者、初创团队和独立开发者。
它的定位类似 Vercel、Docker 与微服务编排平台的结合体。

官网：[https://zeabur.com](https://zeabur.com)

核心理念：

> 让部署变得像写代码一样简单。

## 是否支持 Docker Compose？

部分支持。

目前 Zeabur 支持单个服务使用 Dockerfile 部署，但尚不支持直接上传完整的 docker-compose.yml 来编排多个容器。

你可以通过 Zeabur Dashboard 创建多个服务（Service）来模拟 Compose 中的多个容器，然后：

- 通过环境变量或内部网络让它们互相通信；
- 每个服务可以独立配置镜像、启动命令、端口等。

> 示例：
>
> - 一个 web 服务使用 Dockerfile 启动前端；
> - 一个 db 服务使用 Zeabur 提供的 PostgreSQL；
> - 一个 worker 服务运行后台任务。

未来计划提供类似 docker-compose 的一键多服务部署能力。

## 适用场景

- 独立开发者快速上线个人项目（网站、API、Bot）。
- 希望以 Dockerfile 部署但不想维护服务器。
- 小团队快速搭建演示或微服务系统。
- 需要同时支持多语言、数据库与后台任务的云环境。
- 期望比 Vercel 更灵活、又比传统 IaaS 更简单的方案。

## 入门方式

<Steps>

  <Step title="选择来源">

  </Step>
